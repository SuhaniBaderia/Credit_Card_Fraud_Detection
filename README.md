# Credit_Card_Fraud_Detection
Machine Learning–based Credit Card Fraud Detection System. Built using Python, Scikit-learn, and Streamlit with Logistic Regression, feature scaling, and undersampling to detect fraudulent transactions.


This project implements a **Credit Card Fraud Detection System using Machine Learning**. The model is built using **Python, Scikit-learn, and Logistic Regression** to classify transactions as legitimate or fraudulent.

An interactive **Streamlit web application** is developed to allow users to input transaction details and instantly receive a fraud prediction.

## Features

* Machine learning model using **Logistic Regression**
* **Data preprocessing and feature scaling** using StandardScaler
* **Undersampling technique** to handle class imbalance in fraud datasets
* Interactive **Streamlit web interface**
* Real-time prediction of fraudulent transactions

## Technologies Used

* Python
* Pandas & NumPy
* Scikit-learn
* Streamlit

## How to Run the Project

1. Install required libraries
2. Run the Streamlit app

```bash
streamlit run app.py
```

The application will open in your browser and allow you to test fraud predictions.

## Dataset

The dataset used in this project can be downloaded from Kaggle:

https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

Download the dataset and place `creditcard.csv` in the project folder before running the application.
